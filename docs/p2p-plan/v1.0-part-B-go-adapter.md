## Your role

You ship the Go layer that translates the existing `iface.LedgerEngine`,
`iface.FederationEngine`, and (partially) `iface.AuditEngine` into chain
calls against Part A's contracts. The orchestrator from v0 doesn't change
visibly — it just gets pointed at a different `Ledger` and `Federation`
implementation at construction time.

You are the **only** orchestrator allowed to edit files under
`core/p2p/eth/`. You **also own a small extension to `iface/`** to add the
credit-line methods (`SetPerClientCredit`, `ConsumeCredit`, `RepayCredit`)
and the audit on-chain poster hook (`SetOnchainPoster`). These iface
extensions are explicitly granted to you for v1.0; Part C consumes them
but doesn't define them.

## Implementation notes per file

### `client.go`

```go
type Client struct {
    rpc    *ethclient.Client
    chain  *big.Int
    addr   ContractAddresses
}
type ContractAddresses struct {
    Registry, Ledger, Federation, Audit common.Address
}
func NewClient(ctx context.Context, rpcURL string, chainID *big.Int, addrs ContractAddresses) (*Client, error)
```

- On `NewClient`, call `eth_chainId` and verify it matches.
- Hold contract bindings cached (avoid re-parsing ABIs per call).

### `signer.go`

- Loads private key from env var or file path.
- Per-signer nonce queue. On revert, re-fetch `nonce`, re-sign, re-submit (bounded retries).
- Concurrency-safe: use a mutex around nonce read + transaction submission.

### `ledger_eth.go`

Implements `iface.LedgerEngine`. Each method maps to a contract call:

| iface method | contract call | gwei→wei conversion |
|--------------|---------------|---------------------|
| `DepositStake(node, amount)` | `Ledger.depositStake{value: weiAmount}()` | multiply by 1e9 |
| `WithdrawStake(node, amount)` | `Ledger.withdrawStake(weiAmount)` | multiply by 1e9 |
| `LockForJob(node, jobID, amount)` | `Ledger.lockForJob(node, jobIDBytes32, weiAmount)` | multiply by 1e9 |
| `ReleaseFromJob(jobID, slashFraction)` | `Ledger.releaseFromJob(jobID, bps)` | slashFraction (0..1 float) → bps (0..10000) |
| `SettleJob(jobID, worker, value, networkBps, poolBps)` | `Ledger.settleJob(jobID, worker, grossWei, poolBps)` | value × 1e9 |
| `GetStake(node)` | `Ledger.getStake(node)` (read) | divide returned wei by 1e9 |
| `NetworkTaxRate()` | reads `Ledger.networkTaxBps` (read) | bps / 10000.0 → float64 |
| `UpdateNetworkTaxRate(newRate)` | `Ledger.updateNetworkTaxBps(bps)` (governance auth) | float × 10000 → bps |
| `InsuranceReserve()` | `Ledger.insuranceReserveWei()` (read) | divide by 1e9 |
| `PayoutFromReserve(to, amount, reason)` | `Ledger.payoutFromReserve(to, weiAmount, keccak256(reason))` | × 1e9 |

Cache strategy: read-mostly fields (network tax rate, reserve) cached
with TTL or event-driven invalidation.

### `federation_eth.go`

Implements the extended `iface.FederationEngine`. Same per-method pattern
as Ledger.

### `audit_eth.go`

Implements **partial** `iface.AuditEngine`:

- The in-memory replay + control law from v0 audit stays.
- A new `OnchainPoster` impl posts finalized outcomes via `Audit.postOutcome`.
- `audit.NewEngine()` from v0 gets a new `SetOnchainPoster(p)` method.

You don't replace v0's audit engine wholesale. You add the on-chain
posting layer.

### `events.go`

Subscribes to:

- `StakeDeposited`, `StakeLocked`, `JobSlashed`, `JobSettled` on Ledger
- `RepUpdated`, `CreditExtended`, `CreditRepaid` on Federation
- `OutcomePosted`, `OutcomeFinalized` on Audit

On each event, invalidate the relevant cache entry. Use go-ethereum's
`SubscribeFilterLogs`.

### `cache.go`

Simple `sync.Map` or `*sync.RWMutex`-protected `map[address]record` per
engine. TTL fallback (e.g., 30 sec) for fields without explicit
invalidation events.

### `units.go`

```go
package eth

import "math/big"

// GwToWei converts gwei (int64) to wei (*big.Int).
func GwToWei(g types.Gwei) *big.Int { return new(big.Int).Mul(big.NewInt(int64(g)), big.NewInt(1e9)) }

// WeiToGw converts wei (*big.Int) to gwei (int64).
// Returns the floor; callers must handle precision loss explicitly.
func WeiToGw(w *big.Int) types.Gwei { ... }

// FloatToBps converts a 0..1 float to basis-points (0..10000).
func FloatToBps(f float64) uint16 { ... }
func BpsToFloat(b uint16) float64 { ... }
```

## Test coverage

- **Unit tests against a `*testing.T`-bound Anvil instance** (via Part D's helper once available; OR via your own `testharness/` inside `eth/` until Part D ships).
