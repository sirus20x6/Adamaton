## Your role

You ship all of the Solidity for v1.0: the four contracts (Registry,
Ledger, Federation, Audit), the UUPS proxy plumbing, the OpenZeppelin
dependency tree, Foundry-based tests, and the deployment scripts for
both local Anvil and Base Sepolia.

You are the **only** orchestrator allowed to edit files under
`core/contracts/`. You are also the source of truth for the **ABIs** that
Part B's Go adapter consumes — commit them early and keep them stable.

## ABI publishing (the boundary with Part B)

When `forge build` completes, ABIs land in `core/contracts/out/<Contract>.sol/<Contract>.json`. Each JSON contains the ABI under the `"abi"` key.

To make Part B's life easier, also **export pure-ABI JSON files** to a stable, checked-in location that Part B's `go generate` directives can point at:

```bash
mkdir -p core/contracts/abi
for c in Registry Ledger Federation Audit; do
    jq '.abi' core/contracts/out/$c.sol/$c.json > core/contracts/abi/$c.json
done
```

Wire this into a `forge build` post-hook or a `Makefile` target so it runs
on every rebuild. Part B will use these files with `abigen` to generate
Go bindings. **Commit `core/contracts/abi/*.json` to git** so Part B
doesn't need a Foundry installation to compile.

## Contract specifications

All four contracts are **UUPS upgradable proxies**. The upgrade authority
is the Gnosis Safe (passed in at initialize time). Storage is laid out
per OpenZeppelin's upgrade-safe patterns (no constructor args, use
`initialize()`, no immutable variables, etc.).

### Registry.sol

Soulbound ERC-721 (non-transferable identity NFTs).

```solidity
contract Registry is ERC721Upgradeable, UUPSUpgradeable, AccessControlUpgradeable {
    bytes32 public constant FEDERATION_ROLE = keccak256("FEDERATION");
    bytes32 public constant UPGRADER_ROLE = keccak256("UPGRADER");

    // Reserved storage slots for v1.1 / v1.2
    mapping(uint256 => bytes32) private _commitmentRoot;        // v1.2: ZK rep commitment
    mapping(uint256 => bytes) private _stealthMetaAddress;      // v1.1: EIP-5564 meta-address

    function initialize(address safe) public initializer;
    function mintIdentity(address node, bytes32 donationProof) external onlyRole(FEDERATION_ROLE) returns (uint256);
    function burnIdentity(uint256 tokenId) external onlyRole(FEDERATION_ROLE);
    function nodeOf(uint256 tokenId) external view returns (address);
    function tokenOf(address node) external view returns (uint256);
    function isMinted(address node) external view returns (bool);

    function commitmentRootOf(uint256 tokenId) external view returns (bytes32);
    function setCommitmentRoot(uint256 tokenId, bytes32 root) external; // restricted; empty in v1.0
    function stealthMetaAddressOf(uint256 tokenId) external view returns (bytes memory);
    function setStealthMetaAddress(uint256 tokenId, bytes calldata meta) external; // restricted; empty in v1.0

    // Soulbound: every transfer reverts (override _update, _transfer, approve, etc.)

    function _authorizeUpgrade(address) internal override onlyRole(UPGRADER_ROLE);
}
```

### Ledger.sol

Stake escrow, slashing, settlement, network tax, insurance reserve. All
amounts in wei (uint256). Conversion to/from gwei (int64) happens in Go.

```solidity
contract Ledger is UUPSUpgradeable, AccessControlUpgradeable, ReentrancyGuardUpgradeable {
    bytes32 public constant ORCHESTRATOR_ROLE = keccak256("ORCHESTRATOR");
    bytes32 public constant AUDIT_ROLE = keccak256("AUDIT");
    bytes32 public constant GOVERNANCE_ROLE = keccak256("GOVERNANCE");
    bytes32 public constant UPGRADER_ROLE = keccak256("UPGRADER");

    uint16 public constant NETWORK_TAX_FLOOR_BPS = 50;
    uint16 public constant NETWORK_TAX_CEILING_BPS = 500;
    uint16 public constant MAX_COMBINED_TAX_BPS = 5000;

    struct LockRec { address node; uint256 weiAmount; }
    mapping(address => uint256) public availableWei;
    mapping(bytes32 => LockRec) public locks;       // jobId → lock
    uint256 public insuranceReserveWei;
    uint16 public networkTaxBps;

    function initialize(address safe, address orchestratorRoleHolder, address auditRoleHolder) public initializer;

    function depositStake() external payable;
    function withdrawStake(uint256 weiAmount) external nonReentrant;
    function getStake(address node) external view returns (uint256 available, uint256 lockedTotal);

    function lockForJob(address node, bytes32 jobId, uint256 weiAmount) external onlyRole(ORCHESTRATOR_ROLE);
    function releaseFromJob(bytes32 jobId, uint16 slashFractionBps) external onlyRole(AUDIT_ROLE) returns (uint256 slashed, uint256 returned);
    function settleJob(bytes32 jobId, address worker, uint256 grossWei, uint16 poolTaxBps) external onlyRole(AUDIT_ROLE) returns (uint256 paidWei);

    function payoutFromReserve(address to, uint256 weiAmount, bytes32 reasonHash) external onlyRole(GOVERNANCE_ROLE) nonReentrant;
    function updateNetworkTaxBps(uint16 newBps) external onlyRole(GOVERNANCE_ROLE);

    event StakeDeposited(address indexed node, uint256 weiAmount);
    event StakeWithdrawn(address indexed node, uint256 weiAmount);
    event StakeLocked(address indexed node, bytes32 indexed jobId, uint256 weiAmount);
    event JobSlashed(bytes32 indexed jobId, uint256 slashedWei, uint16 slashFractionBps);
    event JobSettled(bytes32 indexed jobId, address indexed worker, uint256 paidWei, uint256 networkTaxWei, uint256 poolTaxWei);
    event ReservePayout(address indexed to, uint256 weiAmount, bytes32 reasonHash);

    function _authorizeUpgrade(address) internal override onlyRole(UPGRADER_ROLE);
}
```

Rounding policy: when splitting tax + payment, compute one portion via
`mulDiv(grossWei, taxBps, 10000)` and derive the others via subtraction
so the sum is exact (no dust).

### Federation.sol

Pools, reputation, credit lines.

```solidity
contract Federation is UUPSUpgradeable, AccessControlUpgradeable {
    bytes32 public constant AUDIT_ROLE = keccak256("AUDIT");
    bytes32 public constant ORCHESTRATOR_ROLE = keccak256("ORCHESTRATOR");
    bytes32 public constant UPGRADER_ROLE = keccak256("UPGRADER");
    bytes32 public constant GENESIS_ROLE = keccak256("GENESIS"); // one-shot

    int256 public constant SOLO_REP_CAP = 100;
    uint32 public constant DEFAULT_BURN_THRESHOLD = 10;

    struct PoolRec {
        address operator;
        uint16 taxBps;
        uint8 tier;
        bool kycRequired;
        uint256 totalStake;
        uint256 volume;
    }
    struct RepRec {
        int256 earnedScore;          // signed; can go negative
        uint256 donatedVerified;     // wei
        int256 poolEndorsement;
        uint32 defaultCount;
    }
    struct CreditPolicy {
        uint256 perNodeMaxWei;
        uint256 totalExposureWei;
    }

    mapping(bytes32 => PoolRec) public pools;
    mapping(address => bytes32) public poolOf;
    mapping(address => RepRec) public reputations;
    mapping(address => CreditPolicy) public creditPolicies;
    mapping(address => uint256) public outstandingExposure;
    mapping(address => mapping(address => uint256)) public debtWei; // creditor → debtor → wei

    uint256 public minIdentityCreationDonationWei;

    function initialize(address safe, address registry) public initializer;

    function bootstrapGenesis(address founder, int256 premineEarnedScore, uint256 premineDonatedWei) external onlyRole(GENESIS_ROLE); // ONE-SHOT
    function registerPool(uint16 taxBps, bool kycRequired) external returns (bytes32 poolId);
    function joinPool(bytes32 poolId) external;
    function leavePool() external;
    function poolTaxBps(bytes32 poolId) external view returns (uint16);

    function addDonationCredit(address node, uint256 weiAmount) external onlyRole(AUDIT_ROLE);
    function onAuditOutcome(bytes32 jobId, address worker, bool pass, int16 severityBps) external onlyRole(AUDIT_ROLE);
    function effectiveReputationCap(address node) external view returns (int256);
    function defaultCount(address node) external view returns (uint32);

    function setPerClientCredit(uint256 perNodeMaxWei, uint256 totalExposureWei) external;
    function consumeCredit(address creditor, address debtor, uint256 weiAmount) external onlyRole(ORCHESTRATOR_ROLE) returns (bool);
    function repayCredit(address creditor, address debtor, uint256 weiAmount) external onlyRole(ORCHESTRATOR_ROLE);

    event RepUpdated(address indexed node, int256 newScore, uint32 newDefaultCount);
    event CreditExtended(address indexed creditor, address indexed debtor, uint256 weiAmount);
    event CreditRepaid(address indexed creditor, address indexed debtor, uint256 weiAmount);
    event PoolRegistered(bytes32 indexed poolId, address indexed operator);

    function _authorizeUpgrade(address) internal override onlyRole(UPGRADER_ROLE);
}
```

Enforce on-chain: `setPerClientCredit` reverts if `perNodeMaxWei * 2 > minIdentityCreationDonationWei` (so the Sybil math is self-correcting).

### Audit.sol

Outcome posting + 24h dispute window + finalize.

```solidity
contract Audit is UUPSUpgradeable, AccessControlUpgradeable {
    bytes32 public constant VERIFIER_ROLE = keccak256("VERIFIER");
    bytes32 public constant UPGRADER_ROLE = keccak256("UPGRADER");

    struct Outcome {
        bytes32 jobId;
        address worker;
        uint16 similarityBps;
        bool pass;
        int16 severityBps;
        uint64 postedAt;
        uint64 finalizesAt;
    }
    mapping(bytes32 => Outcome) public outcomes;
    mapping(bytes32 => bool) public finalized;
    uint64 public constant DISPUTE_WINDOW = 24 hours;
    uint8 public verifierQuorum; // M of N

    function initialize(address safe, address ledger, address federation, uint8 quorum) public initializer;

    function postOutcome(
        bytes32 jobId, address worker,
        uint16 similarityBps, bool pass, int16 severityBps,
        address[] calldata verifiers, bytes[] calldata sigs
    ) external;

    function challengeOutcome(bytes32 jobId, bytes calldata proof) external;
    function finalize(bytes32 jobId) external; // anyone can call after window

    event OutcomePosted(bytes32 indexed jobId, address indexed worker, bool pass, uint16 similarityBps);
    event OutcomeChallenged(bytes32 indexed jobId, address challenger);
    event OutcomeFinalized(bytes32 indexed jobId);

    function _authorizeUpgrade(address) internal override onlyRole(UPGRADER_ROLE);
}
```

- `postOutcome` requires ≥ `verifierQuorum` valid ECDSA signatures from
  `VERIFIER_ROLE` holders over `keccak256(abi.encode(jobId, worker, similarityBps, pass, severityBps, postedAt))`.
- `finalize` calls Ledger.releaseFromJob → settleJob → Federation.onAuditOutcome → Federation.repayCredit in that order.

## Deployment scripts

`script/Deploy.s.sol` deploys all four contracts via proxies in a single
broadcast. Parameters (Safe address, quorum, initial network tax,
minDonation) are read from env vars or script args.

`script/DeployAnvil.s.sol` is a thin wrapper that uses a stub Safe
(the deploying EOA acts as Safe in tests), and sensible defaults.

`script/DeploySepolia.s.sol` reads the real Safe address from
`SAFE_ADDRESS` env var and uses Base Sepolia chain ID 84532.

`script/BootstrapGenesis.s.sol` is run AFTER `Deploy.s.sol` and calls:
1. `Federation.bootstrapGenesis(founder, premineScore, premineDonatedWei)`
2. `Registry.mintIdentity(founder, bytes32(0))` (genesis path)
3. `Federation.setPerClientCredit(perNodeMaxWei, totalExposureWei)`
4. `Ledger.depositStake{value: workingStakeWei}()`

## Test coverage targets

`forge test` should cover at minimum:

- Every public state-changing method has a happy-path test
- Every revert path (auth, math, reentrancy)
- The Integration.t.sol e2e: deposit → lockForJob → execute (off-chain
  mocked) → postOutcome → finalize → assert (settle paid, tax routed,
  rep updated, credit consumed/repaid)
- Bootstrap genesis: ensure it can only fire once; assert post-conditions
- Sybil math: `setPerClientCredit` reverts if `perNodeMaxWei × 2 > minDonation`
- Slashing: severity = 0 → 0; severity = 1.0 → 10000 bps; partial values map per the v0 curve

Aim for ≥ 95% line coverage. Use `forge coverage`.
